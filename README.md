# geisten 👻

**Local AI that belongs to you.** We build a tiny CPU-only inference engine and the
agents, assistants, and appliances that run on it — private, offline, on hardware
you already own, down to a Raspberry Pi.

## Projects

| Project | What it is |
|---|---|
| [geistlib](https://github.com/geisten/geistlib) | The core: a small, dependency-free AI inference engine in C — CPU-only, local by default |
| [sporegeist](https://github.com/geisten/sporegeist) | Governed agent runtime — policy-gated, budget-bounded, auditable agent actions in pure C23 |
| [geistwissen](https://github.com/geisten/geistwissen) | Local knowledge assistant, links geistlib in-process — no GPU, no cloud, no Python |
| [geist-home-assistant](https://github.com/geisten/geist-home-assistant) | Private local conversation agent for Home Assistant |
| [homebrew-tap](https://github.com/geisten/homebrew-tap) | `brew tap geisten/tap` — install packages on macOS/Linux |
